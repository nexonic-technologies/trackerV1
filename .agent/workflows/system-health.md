# System Health Workflow

> **Purpose**: Aggregate data across ALL audited modules for system-wide risk analysis.

## Steps

1. Scan all audit data in `knowledge_brain/`
2. Scan pattern library for systemic patterns
3. Generate risk heatmap with module breakdown
4. Identify unaudited modules and estimate risk
5. Generate recommendations

## Report includes:
- Module Risk Heatmap (🔴/🟡/🟢)
- Category Distribution (system-wide)
- Systemic Patterns (found in 3+ modules)
- Unaudited Modules with risk estimates
- Next-step recommendations
